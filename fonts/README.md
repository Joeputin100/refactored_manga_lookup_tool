# Fonts Directory

This directory contains Unicode fonts for the label generator to ensure proper Unicode character support on Streamlit Cloud.

## Font Files

- `DejaVuSans.ttf` - Unicode font for proper character rendering
- `LiberationSans-Regular.ttf` - Alternative Unicode font

## Usage

The label generator will use these fonts when Unicode characters are detected in library identifiers or other text fields.
